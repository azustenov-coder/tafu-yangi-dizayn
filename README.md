# Toshkent Amaliy Fanlar Universiteti Portali

Toshkent Amaliy Fanlar Universiteti (TAFU) uchun zamonaviy va funksional veb-portal.

## Loyiha haqida
Ushbu portal universitet yangiliklari, qabul jarayonlari, ilmiy faoliyat va xalqaro hamkorliklar haqida ma'lumot beradi.

## Texnologiyalar
- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **Backend:** Node.js, Express, TypeScript

## Ishga tushirish (Local)

1. Kutubxonalarni o'rnating:
   ```bash
   npm install
   ```

2. Loyihani ishga tushiring (Frontend va Backend):
   ```bash
   npm run dev:all
   ```

3. Brauzerda oching:
   - Frontend: [http://localhost:3000](http://localhost:3000)
   - Backend: [http://localhost:5000](http://localhost:5000)

## Loyiha tuzilmasi
- `backend/` - Node.js backend server
- `components/` - React komponentlari
- `public/` - Statik fayllar va rasmlar
- `App.tsx` - Asosiy routing va ilova tuzilmasi
